Apa itu HTML ?

HTML adalah singkatan dari (HyperText Markup Language), yaitu bahasa (aturan) standar yang digunakan untuk menampilkan teks, gambar video dan audio ke dalam halaman web. HTML merupakan file teks yang tersusun atas elemen elemen yang disebut dengan tag. Tag HTML diapit dengan tana lebih kecil (<) dan tanda lebih besar (>), misalnya: <html>, <head>, <body>, <p>, dan lain lain. Tag HTML ada yang memiliki pasangan, ada juga yang tidak. Jika sutau tag memiliki pasangan, maka tag penutup akan disertai dengan tanda slash (/), misalnya </html>, </head>, </body>, </p>, dan lain lain.

Dokumen atau file HTML dapat dibuat menggunakan aplikasi Text Editor apa saja, dan disimpan dengan eksitensi .html atau .htm. Untuk memudahkan proses belajar anad, penulis merekomendasikan agar anda menggunakan Text Editor yang secara khusus dirancang untuk menulis kode-kode HTML. Saat ini banyak Software Text Editor untuk HTML yang ditawarkan secara gratis di internet.
