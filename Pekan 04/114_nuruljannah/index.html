Apa itu HTML ?

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk menampilkan teks, gambar, video, dan audio di halaman web. HTML terdiri dari elemen-elemen yang disebut tag. Contoh tag: <html>, <head>, <body>, <p>.

Dokumen HTML bisa dibuat dengan aplikasi Text Editor apa saja dan disimpan dalam format .html atau .htm. Sebaiknya gunakan Text Editor yang dirancang khusus untuk kode HTML.
