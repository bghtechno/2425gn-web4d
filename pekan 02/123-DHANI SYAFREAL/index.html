Apa Itu HTML

HTML adalah singkatan dari (Hyper Text arkup Language), yaitu bahasa (aturan) standar yang digunakan untuk menampilkan teks, gambar, video dan audio ke dalam halaman web. HTML merupakan <1>file teks yang tersusun atas elemen-elemen yang disebut dengan <1>tag. <1>Tag HTML diapit dengan tanda lebih kecil (<) dan tanda lebih besar (>), misalnya: lthtmlsgt, sitheadsgt, ltbodysgt, sitpsgt, dan lain-lain. tag memiliki pasangan, maka <1>tag penutup akan disertai dengan tanda slash (/), misalnya: slt/htmlsgt, slt/headsgt, slt/bodysgt, slt/pagt, dan lain-lain.

Dokumen atau <1>file HTML dapat dibuat menggunakan aplikasi <1>Text Editor apa saja, dan disimpan dengan eksitensi .html atau .htm. Untuk memudahkan proses belajar Anda, penulis merekomendasikan agar Anda menggunakan Text Editor yang secara khusus dirancang untuk menulis kode-kode HTML. Saat ini banyak <1>software Text Editor untuk HTML yang ditawarkan secara gratis di Internet.
